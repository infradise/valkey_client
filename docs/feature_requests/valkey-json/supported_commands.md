## Supported JSON Commands (Redis and Valkey)

### Commands Supported Only by Redis JSON
```sh
JSON.MERGE
```

### Commands Supported Only by Valkey JSON (aka. valkey-json)
```
JSON.DEBUG DEPTH
JSON.DEBUG FIELDS
JSON.DEBUG HELP
```

### JSON Commands Supported by Both Redis and Valkey

```sh
JSON.ARRAPPEND
JSON.ARRINDEX
JSON.ARRINSERT
JSON.ARRLEN
JSON.ARRPOP
JSON.ARRTRIM
JSON.CLEAR
JSON.DEBUG
JSON.DEBUG MEMORY 
JSON.DEL
JSON.FORGET
JSON.GET
JSON.MGET
JSON.MSET
JSON.NUMINCRBY
JSON.NUMMULTBY
JSON.OBJKEYS
JSON.OBJLEN
JSON.RESP
JSON.SET
JSON.STRAPPEND
JSON.STRLEN
JSON.TOGGLE
JSON.TYPE
```
